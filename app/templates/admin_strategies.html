Tips

• The Strategy ID must match exactly what your TradingView alert sends in "strategy_id" — case-sensitive.
• Hyperliquid symbols are bare tickers: BTC, ETH, SOL. Bybit USDT-perp symbols are BTCUSDT, ETHUSDT, etc.
• Enabled = false keeps the config but skips orders — useful for pausing a strategy during regime change without losing the row.
• Changes are atomic: file write + router reload happen together. If the next webhook fires within a second of saving, it uses the new config.
• If you ever leak the webhook secret, rotate via fly secrets set WEBHOOK_SECRET=$(openssl rand -hex 32) then update your TradingView alert body too.
